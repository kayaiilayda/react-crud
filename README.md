# İlayFlix

Bu uygulama, responsive uyumlu biçimde bir dizi-film uygulamasıdır.

## Temel Özellikleri

- Dizi/Film Eklenebilir
- Dizi/Film Güncellenebilir
- Dizi/Film Silinebilir
- Dizi/Film Listelenebilir
- Dizi/Film tipine bağlı olarak filtrelenebilir
